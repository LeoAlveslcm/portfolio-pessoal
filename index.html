HTML é o componente base da web. Ele permite a construção de websites e a inserção de novos conteúdos por meio de hipertextos.

CSS é uma linguagem de estilos utilizada para definir cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

Javascript, ou simplesmente JS, é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web.

Bootstrap é um framework front-end que fornece estruturas de CSS para criação de sites e aplicações responsivas de forma rápida e simples.
